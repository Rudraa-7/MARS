A machine learning and NLP-based web application that detects distress messages from textual communication during war-time or emergency situations. The system classifies messages as Distress or Non-Distress using ML algorithms such as Naïve Bayes, SVM, and Logistic Regression with TF-IDF vectorization.
# 🔥 Key Features of the War-Time Distress Signal Detection System

🚨 Real-Time Distress Detection — Instantly analyzes messages and classifies them as Distress or Non-Distress.
🧠 Machine Learning Powered — Uses ML algorithms such as Naïve Bayes, SVM, and Logistic Regression for intelligent prediction.
🌍 Multilingual Support — Detects distress signals in multiple languages including English, Hindi, Russian, and Japanese.
🔑 Special Keyword Detection (“MARS”) — Any message containing the keyword “mars” is automatically classified as a Distress Signal.
📝 Text-Based Analysis — Users can directly enter messages through the web interface for instant classification.
🎙️ Audio Input Support — Supports conversion of speech/audio into text for distress analysis.
🔴 Live Voice Recording — Enables live recording and real-time distress signal monitoring.
⚡ Fast TF-IDF Feature Extraction — Converts textual communication into machine-readable vectors for efficient analysis.
🌐 Web-Based User Interface — Built using Flask, HTML, and CSS for easy accessibility and real-time interaction.
📊 Prediction Logging & Monitoring — Maintains records of analyzed messages and generated alerts.
🛡️ War-Time Emergency Focused — Designed specifically for crisis communication, emergency response, and battlefield distress monitoring.
